## Description

This repository provides a simple and reproducible Docker-based setup to run a WordPress website with a MariaDB database.

## Quickstart

### Prerequisites

- Docker
- Docker Compose

### Start the services

```bash
docker-compose up -d
```

### Visit your WordPress locally:
```bash
http://localhost:8080
```
### on your cloud VM:
```bash
http://your_cloud_vm:8080
```
